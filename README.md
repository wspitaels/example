# example

## Hacked container simulator (harmless)

Run the script below for a theatrical, colorful terminal animation that **simulates** a hacked container. It does not change files, install anything, or run destructive commands.

### Run directly from GitHub with curl

Use the raw file URL (not the normal GitHub page URL):

```bash
curl -fsSL https://raw.githubusercontent.com/<owner>/<repo>/main/hacked-container-sim.sh | bash
```

Example:

```bash
curl -fsSL https://raw.githubusercontent.com/wspitaels/example/main/hacked-container-sim.sh | bash
```

```bash
chmod +x hacked-container-sim.sh
./hacked-container-sim.sh
```

Press `Ctrl+C` to stop.
